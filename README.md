 Sistema de Cálculo de Corrida de Táxi (PHP)
 Descrição do Projeto
 
Este projeto foi desenvolvido no âmbito da formação
 Capacita - CFTI (Centro de Formação em Tecnologias de Informação), com o objetivo de aplicar os conceitos fundamentais da linguagem PHP no desenvolvimento de uma solução simples e funcional.
O sistema simula o cálculo do preço de uma corrida de táxi por aplicativo, considerando fatores como distância percorrida e aplicação de taxa adicional em fins de semana.

 Objetivos

 Funcionalidades
Cálculo automático do valor da corrida
Aplicação de taxa adicional de 5% em fins de semana
 Exibição detalhada das informações da corrida
Organização modular do código (funções e procedimentos)

 Dados Considerados
O sistema utiliza os seguintes dados:
•	Nome do motorista
•	Nome do passageiro
•	Local de origem
•	Destino
•	Distância (em km)
•	Indicação de fim de semana (true/false)

 Lógica de Cálculo
•	Tarifa fixa por quilômetro (definida como constante)
•	Fórmula base: Valor = Distância × Tarifa_por_KM
•	Se for fim de semana: Valor Final = Valor + (Valor × 5%)
